# Contexte

L'entreprise dispose d'un serveur de base de données contenant une table de clients.
Deux profils utilisateurs accèdent à cette base :
- un opérateur ;
- un responsable de centre d'appels.

L'objectif du TP est de :
- consulter les données ;
- tester les droits d'accès ;
- simuler une suppression ;
- analyser la traçabilité des actions et la responsabilité associée.
